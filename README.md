# 中医药知识图谱

一个基于 D3.js 的交互式中医药知识图谱可视化项目。

## 项目结构

```
knowledge-graph/
├── index.html              # 主可视化页面（包含嵌入数据）
├── herbs.json              # 药材数据（396 味中药）
├── formulas.json           # 方剂数据（200 个经典方剂）
├── relationships.json      # 关系数据（982 条药材-方剂关系）
├── herbs-data.js           # 药材数据（JavaScript 格式）
├── formulas-data.js        # 方剂数据（JavaScript 格式）
└── relationships-data.js   # 关系数据（JavaScript 格式）
```

## 功能特性

- **交互式图谱**：基于 D3.js 力导向图，支持缩放、拖拽、平移
- **节点类型**：
  - 🟢 药材节点（按分类着色）
  - 🔵 方剂节点（方形，更大尺寸）
- **关系展示**：药材与方剂之间的组成关系（君臣佐使）
- **搜索功能**：实时搜索药材或方剂名称
- **筛选功能**：按类型（全部/药材/方剂）筛选显示
- **详情卡片**：点击节点查看详细信息（功效、主治、组成等）
- **高亮关联**：悬停或点击节点时高亮相关节点和关系
- **图例说明**：底部右侧显示药材分类颜色图例

## 使用方法

### 直接打开

双击 `index.html` 文件在浏览器中打开，即可使用完整功能（数据已嵌入 HTML 中）。

### 本地服务器（可选）

如果需要通过网络访问，可以使用简单的 HTTP 服务器：

```bash
# Python 3
python3 -m http.server 8000

# 然后访问 http://localhost:8000
```

## 数据说明

### 药材数据（herbs.json）

包含 396 味中药，每味药包含：
- `id`: 唯一标识符（H001-H396）
- `name`: 药材名称
- `category`: 分类（如：解表药-发散风寒）
- `nature`: 性味（如：辛、微苦，温）
- `meridian`: 归经（如：肺、膀胱经）
- `effect`: 功效
- `indication`: 主治
- `source`: 来源典籍
- `source_url`: 百度百科链接

### 方剂数据（formulas.json）

包含 200 个经典方剂，每个方剂包含：
- `id`: 唯一标识符（F001-F200）
- `name`: 方剂名称
- `category`: 分类（如：解表剂-辛温解表）
- `effect`: 功效
- `indication`: 主治
- `composition`: 组成（药材列表，包含 herb、role、dosage）
- `source`: 来源典籍
- `source_url`: 百度百科链接

### 关系数据（relationships.json）

包含 982 条药材-方剂关系，每条关系包含：
- `id`: 唯一标识符
- `source`: 药材 ID
- `target`: 方剂 ID
- `herb`: 药材名称
- `formula`: 方剂名称
- `role`: 角色（君、臣、佐、使）
- `dosage`: 剂量

## 技术方案

- **前端**：D3.js v7（力导向图可视化）
- **数据格式**：JSON（标准格式）+ JavaScript 变量（备用）
- **部署**：单文件 HTML（数据嵌入，无需服务器）

## 操作指南

1. **缩放**：鼠标滚轮
2. **平移**：按住空白区域拖拽
3. **拖拽节点**：按住节点拖拽
4. **查看详情**：点击节点
5. **高亮关联**：悬停节点
6. **搜索**：在顶部搜索框输入关键词
7. **筛选**：使用顶部按钮筛选节点类型
8. **关闭详情**：点击详情卡片右上角 × 或点击空白区域

## 扩展说明

### 添加新药材

编辑 `herbs.json`，添加新药材对象，格式参考现有数据。

### 添加新方剂

编辑 `formulas.json`，添加新方剂对象，并在 `relationships.json` 中添加相应的药材-方剂关系。

### 重新生成可视化

运行 `build_html.py` 脚本重新生成 `index.html`：

```bash
python3 build_html.py
```

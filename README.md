# 中医药知识图谱

一个基于 D3.js 的交互式中医药知识图谱可视化项目，包含 396 味药材、200 个经典方剂及其关系。

## 🎯 项目简介

本项目旨在构建一个完整的中医药知识图谱，通过可视化方式展示药材与方剂之间的关联关系，帮助用户直观理解中医药的配伍理论。

### 核心数据

- **396 味中药材** - 涵盖解表药、清热药、补虚药等 28 个分类
- **200 个经典方剂** - 从《伤寒论》到《温病条辨》的经典名方
- **982 条关系** - 药材与方剂之间的组成关系（君臣佐使）

## ✨ 功能特性

### 核心功能

- **交互式图谱**：基于 D3.js 力导向图，支持缩放、拖拽、平移
- **节点类型**：
  - 🟢 药材节点（按分类着色，28 种颜色）
  - 🔵 方剂节点（方形，更大尺寸）
- **关系展示**：药材与方剂之间的组成关系（君臣佐使）
- **搜索功能**：实时搜索药材或方剂名称
- **筛选功能**：按类型（全部/药材/方剂）筛选显示

### 高级功能

- **数据统计面板**：左上角显示数据统计（药材总数、方剂总数、关系总数、分类统计）
- **药材分类筛选**：按分类（如解表药、清热药）筛选药材节点
- **方剂功效筛选**：按功效分类筛选方剂节点
- **节点群集模式**：按分类群集药材节点，点击群集节点显示分类详情
- **路径分析**：查找两种药材之间通过方剂的最短关联路径

### 交互体验

- **详情卡片**：点击节点查看详细信息（功效、主治、组成、来源等）
- **高亮关联**：悬停或点击节点时高亮相关节点和关系
- **图例说明**：底部右侧显示药材分类颜色图例
- **响应式设计**：支持缩放、拖拽、平移等交互操作

## 🚀 使用方法

### 方法一：直接打开（推荐）

双击 `index.html` 文件在浏览器中打开，即可使用完整功能（数据已嵌入 HTML 中）。

**注意**：需要网络连接加载 D3.js 库（CDN）。

### 方法二：本地服务器

如果需要通过网络访问，可以使用简单的 HTTP 服务器：

```bash
# 进入项目目录
cd /Users/sai/WorkBuddy/中医/knowledge-graph

# Python 3
python3 -m http.server 8000

# 然后访问 http://localhost:8000
```

## 📁 项目结构

```
knowledge-graph/
├── index.html              # 主可视化页面（380KB，包含嵌入数据）
├── README.md               # 项目文档
├── build_html.py           # HTML 生成脚本
├── herbs.json              # 药材数据（396 味）
├── formulas.json           # 方剂数据（200 个）
├── relationships.json      # 关系数据（982 条）
├── herbs-data.js           # 药材数据（JavaScript 格式）
├── formulas-data.js        # 方剂数据（JavaScript 格式）
└── relationships-data.js   # 关系数据（JavaScript 格式）
```

### 文件说明

- **index.html**：主可视化页面，所有数据已嵌入，可直接在浏览器中打开使用
- **build_html.py**：Python 脚本，用于重新生成 index.html（修改数据后运行）
- **\*.json**：标准 JSON 格式的数据文件，可被其他程序调用
- **\*-data.js**：JavaScript 格式的数据文件，供网页直接引用

## 📊 数据说明

### 药材数据（herbs.json）

包含 396 味中药，每味药包含：

| 字段 | 说明 | 示例 |
|------|------|------|
| `id` | 唯一标识符 | H001 |
| `name` | 药材名称 | 麻黄 |
| `category` | 分类 | 解表药-发散风寒 |
| `nature` | 性味 | 辛、微苦，温 |
| `meridian` | 归经 | 肺、膀胱经 |
| `effect` | 功效 | 发汗解表，宣肺平喘，利水消肿 |
| `indication` | 主治 | 风寒感冒，胸闷喘咳，风水浮肿 |
| `source` | 来源典籍 | 《神农本草经》 |
| `source_url` | 百度百科链接 | https://baike.baidu.com/item/... |

### 方剂数据（formulas.json）

包含 200 个经典方剂，每个方剂包含：

| 字段 | 说明 | 示例 |
|------|------|------|
| `id` | 唯一标识符 | F001 |

| `effect` | 功效 | 发汗解表，宣肺平喘 |
| `indication` | 主治 | 风寒表实证 |
| `composition` | 组成 | [{herb: "麻黄", role: "君", dosage: "9g"}, ...] |
| `source` | 来源典籍 | 《伤寒论》 |
| `source_url` | 百度百科链接 | https://baike.baidu.com/item/... |

### 关系数据（relationships.json）

包含 982 条药材-方剂关系，每条关系包含：

| 字段 | 说明 | 示例 |

| `target` | 方剂 ID | F001 |
| `herb` | 药材名称 | 麻黄 |
| `formula` | 方剂名称 | 麻黄汤 |
| `role` | 角色 | 君 |

## 🖱️ 操作指南

### 基础操作

1. **缩放**：鼠标滚轮或双指捏合
2. **平移**：按住空白区域拖拽
3. **拖拽节点**：按住节点拖拽
4. **查看详情**：点击节点
5. **高亮关联**：悬停节点
6. **搜索**：在顶部搜索框输入关键词
7. **筛选**：使用顶部按钮筛选节点类型
8. **关闭详情**：点击详情卡片右上角 × 或点击空白区域

### 高级功能

9. **数据统计**：查看左上角统计面板
10. **药材分类筛选**：使用工具栏下拉框按分类筛选
11. **方剂功效筛选**：使用工具栏下拉框按功效筛选
12. **群集模式**：点击"群集模式"按钮，按分类群集节点
13. **路径分析**：点击"路径分析"按钮，查找药材间的最短路径

## 🔧 扩展说明

### 添加新药材

1. 编辑 `herbs.json`，添加新药材对象：
   ```json
   {
     "id": "H397",
     "name": "新药材",
     "category": "补虚药-补气",
     "nature": "甘，温",
     "meridian": "脾、肺经",
     "effect": "功效描述",
     "indication": "主治描述",
     "source": "来源典籍",
     "source_url": "https://baike.baidu.com/item/..."
   }
   ```

2. 运行 `build_html.py` 重新生成 `index.html`：
   ```bash
   python3 build_html.py
   ```

### 添加新方剂

1. 编辑 `formulas.json`，添加新方剂对象：
   ```json
   {
     "id": "F201",
     "name": "新方剂",
     "category": "解表剂-辛温解表",
     "effect": "功效描述",
     "indication": "主治描述",
     "composition": [
       {"herb": "药材1", "role": "君", "dosage": "10g"},
       {"herb": "药材2", "role": "臣", "dosage": "6g"}
     ],
     "source": "来源典籍",
     "source_url": "https://baike.baidu.com/item/..."
   }
   ```

2. 编辑 `relationships.json`，添加相应的药材-方剂关系

3. 运行 `build_html.py` 重新生成 `index.html`

### 修改样式或功能

1. 修改 `build_html.py` 脚本
2. 运行脚本重新生成 `index.html`：
   ```bash
   python3 build_html.py
   ```
